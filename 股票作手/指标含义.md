# 指标含义

| 指标 | 含义解析                                                     |
| ---- | ------------------------------------------------------------ |
| MA5  | 5天均线                                                      |
| MA10 | 10天均线                                                     |
| MA20 | 20天均线                                                     |
| MA60 | 60天均线                                                     |
| BOLL | 布林线（Boll）：该指标在图形上画出三条线，其中上下两条线可以分别看成是股价的压力线和支撑线，而在两条线之间还有一条股价平均线，布林线指标的参数最好设为20。一般来说，股价会运行在压力线和支撑线所形成的通道中。 |
